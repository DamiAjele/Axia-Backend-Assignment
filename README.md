# Express User App
This is a simple express app that allows users to register and log in

## Features
- User Registration
- User Login

## API Endpoints
### For Registration
- **POST** `http://localhost:5000/`

### For Login
- **POST** `http://localhost:5000/user-login`

### Other Endpoints
- **PUT** `http://localhost:5000/`
- **DELETE** `http://localhost:5000/`

## Dependencies
- express
- mongoose
- bcrypt.js
- dotenv
- nodemon
